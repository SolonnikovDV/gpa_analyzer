{# Модальное окно «Подсказки» для страницы детального анализа #}
Подсказки
  • DDL функции — вставьте полный текст CREATE OR REPLACE FUNCTION ... AS $function$ ... $function$, включая тело.
  • Параметры — на шаге настройки размеров таблиц укажите параметры в том же порядке, что и в сигнатуре функции. Для строк используйте кавычки, например: '2024-01-01', 'current_date', '5', ''.
  • Желаемое количество строк — можно оставить текущее или задать своё для оценки нагрузки при других объёмах данных.
  • Подключение к БД — опционально. Если включено, статистика таблиц подтягивается из базы; иначе используются значения по умолчанию.
