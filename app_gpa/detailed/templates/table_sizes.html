Настройка размеров таблиц и параметров функции
Важно! Укажите параметры функции в том же порядке, что и в сигнатуре. Они будут использованы для подстановки в запросы.
Введите параметры через запятую. Для строковых значений используйте кавычки. Например: '2024-01-01', 'current_date', '5', ''
Найденные таблицы (укажите желаемое количество строк):
Таблица Текущее строк Текущий размер (GB) Средняя ширина строки (байт) Желаемое строк
Примечание: Оставьте желаемое количество строк равным текущему, если не хотите изменять размер таблицы. Для пустых таблиц укажите желаемый размер.
